# このリポジトリについて

これは pipenv + jupyter のテンプレートリポジトリです。

# 動作確認済み環境

| 名称   | 条件/バージョン | 備考                                                       |
| ------ | --------------- | ---------------------------------------------------------- |
| Python | 3.8.0 以上      | 　                                                         |
| pipenv | 2022.11.25 以上 | pip でグローバルにインストールする必要あり                 |
| VSCode | 1.74.2 以上     | このプロジェクトの推奨拡張機能もインストールしておくとよい |

# セットアップ方法

clone 後、ターミナルから次のコマンドを実行する。

```bash
bash setup.sh
```
